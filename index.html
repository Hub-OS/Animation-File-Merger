Animation File Merger
GitHub

A utility for merging BoomSheets animation files.

Not affiliated with BoomSheets.

Usage:

Drag and drop BoomSheets `.animation` and `.png` files with matching names into the dashed box.

Click the "Merge" button to generate a image and animation data. Save the final image using the context menu by right clicking on it.
